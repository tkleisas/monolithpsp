Tetris κλώνος v1.0
-----------------
Πρόκειτα για έναν αρκετά απλό κλώνο του tetris. Υλοποιήθηκε σε τρείς διαστάσεις!
Το πρόγραμμα βασίζεται σε ένα παράδειγμα (cube) από το PSPSDK που μπορείτε να βρείτε στη διεύθυνση http://www.ps2dev.org

Έχει δοκιμαστεί σε 3.40-OE firmware, αλλά πρέπει να δουλεύει σε firmware 1.50 και στα περισσότερα OE firmwares.

Εγκατάσταση
------------
για τους χρήστες του 1.50 firmware->αντιγράψτε τους καταλόγους psptetris και psptetris% στο φάκελο PSP/GAME150 
για τους χρήστες OE firmware->απλά αντιγράψτε τον φάκελο psptetris στο φάκελο PSP/GAME

Χειρισμός 
---------
αριστερά              -> μετακίνηση αριστερά
δεξιά                 -> μετακίνηση δεξιά
κάτω                  -> γρήγορη μετακίνηση κάτω
πάνω                  -> περιστροφή
κουμπί Χ              -> περιστροφή
κουμπί τετράγωνο      -> παύση
αναλογικό χειριστήριο -> αλλαγή προοπτικής
κουμπί home           -> έξοδος
κουμπί select         -> επαναφορά προοπτικής
κουμπί start          -> επανεκκίνηση παιχνιδιού

Πράγματα που λείπουν
--------------------
Ήχος
Επιλογή επιπέδου
Πρόσθετα είδη παιχνιδιού
